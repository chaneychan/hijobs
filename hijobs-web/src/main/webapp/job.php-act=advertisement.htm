广告销售高级经理

工作地点：上海市

岗位职责：

  • 1、开发新的广告主及广告代理渠道，持续积累客户数量，客户信息的搜集管理；
  • 2、定期对客户档案进行分析、整理提供销售分析数据；
  • 3、为客户定制互联网广告产品，创新广告，能策划相关的广告营销方案，建立长期合作关系。

任职要求：

  • 1、热爱互联网，有传统媒体或互联网广告销售经验；
  • 2、有1-2年互联网广告销售经验/4A广告公司工作经验，具备敏锐的市场洞察力；
  • 3、能够把握和挖掘客户的需求，并提供解决方案；
  • 4、有一定的策划能力，能独立完成策划案及PPT制作；
  • 5、性格开朗、外向，主动性强，工作态度积极，热爱销售工作，能适应工作压力和敢于面对挑战；
  • 6、有较强的沟通能力，具备优秀的销售和谈判能力，以及良好的客户开拓和服务意识。

华北区高级销售经理

工作地点：北京-海淀区

岗位职责：

  • 1、负责广告客户的开发和维护工作，与各级渠道和直客建立沟通，建立与品牌客户的良好合作关系；
  • 2、与公司内各个关联部门沟通，确保广告的及时投放与问题反馈、广告资源的合理利用与开发；
  • 3、负责实施公司安排的销售计划和目标，完成指标；
  • 4、收集并整理相关的市场及客户信息，及时将信息汇报上级，为公司的业务决策提供信息支持。

任职要求：

  • 1、三年以上主流媒体销售经验，有一年以上互联网销售工作经验；
  • 2、丰富的品牌客户服务经验，有4A渠道、品牌客户（直客）资源者优先考虑；
  • 3、形象好，气质佳，性格开朗、外向，有较强的沟通能力及强烈的成功欲望，可以承受较大工作压力；
  • 4、具备良好的客户开拓、服务能力，优秀的销售、谈判以及口笔表达交流的技巧；
  • 5、有移动互联网销售经验者可优先考虑。

客户助理

工作地点：上海市

岗位职责：

  • 1、统计销售数据，并制作、分析相关报表；
  • 2、收集、整理销售文件及市场资料；
  • 3、根据财务报表跟踪客户回款状况；
  • 4、配合销售人员处理相关后勤工作，并完成销售经理要求的临时性工作。

任职要求：

  • 1、大专及以上学历，专业不限；
  • 2、性格活泼开朗，工作积极主动，沟通能力及责任心强，耐心、细心，能够承受工作压力；
  • 3、了解互联网行业及广告行业，有销售助理或4a／媒体工作经验者优先。
