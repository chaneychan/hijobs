网络管理员

工作地点：广州-天河区

岗位职责：

  • 1、能独立完成系统方案的设计制作并负责方案的验证工作，包括组网方案、IP规划、设备选型等具体工作，研究项目施工的可行性；
  • 2、维护公司计算机系统的稳定运行，对其PC, OA设备，局域网、广域网、电话系统等设施进行管理；
  • 3、为公司提供IT软硬件方面的服务和支持。

任职要求：

  • 1、熟练掌握服务器、交换机、路由器以及存储、备份系统、网络安全产品的规划、配置及调试工作；
  • 2、有良好的沟通、文字编写能力，有较强的客户服务意识及团队合作精神；
  • 3、具有以下认证证书之一，CCNP/HCSE/MCSE/CISP证书/信息系统监理工程师认证者或信息系统项目管理师证书者优先。

备注：该职位往内容运营方向培养。
